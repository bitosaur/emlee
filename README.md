Emlee EML File Viewer
A lightweight and secure tool for viewing .eml email files without an email client.
This also opens .msg files but rather slowly.
The reason this project was created was to Navigate to Previous or Next email on the same folder. 
This is most useful when you have a bunch of emails downloaded or exported from a mailbox into a folder.
